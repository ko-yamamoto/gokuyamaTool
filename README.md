gokuyamaTool
=================

Experimental commandline tool for okuyama

Example
------------

### Set ###

simple set a key and a value

```sh
gokuyamaTool set {key} {value}
```

or set with a tag

```sh
gokuyamaTool set {key} {value} {tag}
```

### Get ###

get a value

```sh
gokuyamaTool get {key}
```

### Deal with Tag ###

get keys from a tag

```sh
gokuyamaTool tag {tag}
```
